Work Experience

Over this past summer (2015), I worked with the physiological startup PhysIQ as a Software Development Intern. The first project I worked on was learning how to use the Java SDK to interact with the RESTful API. The objective of this project was to clean up the existing documentation and to write a user guide from a brand new user's perspective. In the process, I was also able to help find some bugs in the API.

I'm currently facilitating a unit testing transition to behaviorial-driven testing, using Cucumber. There are many SDKs that interact with the API and there are over a thousand unit tests, using Cucumber allows us to consolidate feature files that are both human and machine readable.

Over the summer of 2014, I worked as a research and coding assistant in the Computational Thinking in STEM (CT-STEM) research group at Northwestern University. The goal of the research group was to develop curriculum that challenged high school students to develop programming and coding skills, while learning to think more computationally.

As a coding assistant, I created my own model of the spread of vector-borne diseases in NetLogo, an agent-based modeling language. I also maintained existing models, tailoring them to the specific needs of certain programs as well as debugging models. I helped develop curriculum in the block-based programming language Blockly.

As a research assistant, I helped assess and analyze assessment results from the school test sites. I also contributed to the creation and refinement of the assessments that tracked student progress throughout the school year.
